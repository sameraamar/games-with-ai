Publish On GitHub Pages

In the repository settings, enable Pages and publish from the main branch root. GitHub will serve this file as the homepage at https://YOUR-USERNAME.github.io/YOUR-REPO/.

  1. Push this repository to GitHub.
  2. Open repository settings and go to Pages.
  3. Choose Deploy from a branch.
  4. Select the main branch and the root folder.
  5. Wait for the Pages URL to be generated.
Static hosting only. No server runtime is required for the current games.
